## 🛠️ Main Command Matrix

Use the integrated `Makefile` to manage the core:

| Command | Action |
| :--- | :--- |
| `make dev` | Launch the Sovereign Backend & Portal |
| `make build` | Compile Go, C++, and Frontend |
| `make foundation-gen` | Generate Synthetic Intelligence Data |
| `make foundation-train`| Train your own AI from Scratch |
| `make foundation-chat` | Chat with your Scratch-built AI |
